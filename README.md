## 🎯 Objetivo do Projeto
O objetivo deste projeto é transitar de um modelo tabular relacional para uma **Base de Dados Orientada a Grafos**. Através da linguagem Cypher, procuramos extrair inteligência de negócio sobre microtransações (Shark Cards), transição de consumo físico para digital e benchmarking de mercados globais (com foco em Portugal).
